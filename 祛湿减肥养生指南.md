## 🍵 祛湿茶｜办公室必备

### ✅ 经典祛湿茶配方

**陈皮茯苓茶**（适合所有人）

```
材料：陈皮5g + 茯苓10g + 炒薏米10g
做法：热水冲泡，代茶饮
功效：健脾祛湿，化痰消肿
```

**升级版**（湿热体质）

```
材料：陈皮5g + 茯苓10g + 炒薏米10g + 荷叶5g
做法：热水冲泡
功效：祛湿+清热+减脂
```

> 💡 **饮用时间**：上午9-11点（脾经值班，效果最好）

脾虚 → 水湿滞留 → 形成湿气

所以祛湿的同时**一定要健脾**！
